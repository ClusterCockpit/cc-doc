{{< /blocks/cover >}} {{% blocks/lead color="secondary" %}} ClusterCockpit is a framework for job-specific performance and energy monitoring on HPC clusters with a focus on simple installation and maintenance, high security and intuitive usage. {{% /blocks/lead %}} {{< blocks/section color="dark" type="row" >}} {{% blocks/feature icon="fas fa-server" title="Scalable" %}}

Supports multiple clusters in one web interface. Scales to thousands of nodes and millions of jobs. Supports heterogeneous clusters with and without node sharing.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-wrench" title="Slurm integration" url="https://github.com/ClusterCockpit/cc-slurm-adapter" %}}

Comes with a ready to use Slurm integration. Can be integrated with any batch job scheduler via a REST API.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-display" title="Modern Web UI" url="./docs/webinterface/" %}}

Responsive web interface with job specific, node, user, and system views. Provides metric plots, aggregate statistics, roofline plots and ressource table, and access to job script. Fully user configurable.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-chart-simple" title="Global access to metric data" url="./docs/tutorials/prod-metric-list/" %}}

Access to configurable set of metrics including hardware performance counter data. Comes with powerful HPC centric node agent, but can also be integrated with other node agent solutions.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-lock" title="Authentication methods" url="./docs/reference/cc-backend/authentication/" %}}

Supports local accounts, LDAP, and KeyCloak OpenID Connect. Can be integrated with existing user portals using JWT based authentication.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-users" title="User roles" url="./docs/explanation/roles/" %}}

Supports roles for users, project managers, support personnel, and administrators. Users can only see their own jobs. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-filter" title="Job sorting and filtering" url="./docs/webinterface/filters/" %}}

Powerful sorting of jobs according to all job metadata attributes. Filter for job metadata and aggregate metric data attributes.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-magnifying-glass" title="Unified search bar" url="./docs/webinterface/searchbar/" %}}

A unified search bar allows to search for job ids, job names, project ids, usernames, and names.

{{% /blocks/feature %}} {{% blocks/feature icon="fas fa-tag" title="Job tagging" url="./docs/webinterface/views/tags/" %}}

Jobs can be tagged. Job tags are grouped by type and can have a configurable visibility using a scope attribute.
